## Setup

1. Install Node (v16+ recommended).
2. From the project root run:

```bash
npm install
npx playwright install
```

## Run examples

These files are simple Node scripts. Run an example with:

```bash
node chapter_01_basics/Lab01_basics.js
node chapter_01_basics/Lab02_machineInfo.js
```
